# What is Konfig?

**Konfig makes it easy to create SDKs for APIs so onboarding is seamless for
developers.** We do this by providing a set of tools to turn your OpenAPI spec or
Postman Collection into high quality SDKs in the most popular languages.

<Figure
  caption="Konfig generates SDKs so developers can easily integrate your API"
  src="/img/konfig-workflow.svg"
/>

Today our [CLI](https://www.npmjs.com/package/konfig-cli) can get you started in
minutes and handle any level of customization that you need through simple
configuration.

<Figure
  caption="Konfig solves the three steps of creating SDKs. Preparing to generate (1), Generating SDKs (2), and Maintenance (3)."
  src="/img/konfig-mindmap.svg"
/>
